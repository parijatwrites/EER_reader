# EER_header
#This is a python script for reading the header files of .eer files from Falcon detectors. Use it as you wish. 
#Usage:

python eer_header.py /path/to/eer/file/file.eer

#if your python links to python2.7, run the command using python3 instead.

#This code uses the following modules: tifffile, xmltodict, argparse and sys.

#tifffile and xmltodict can be installed by running the following command:

pip install xmltodict tifffile
